# Форма заявки

Адаптивная форма обратной связи с полями для имени и телефона. При отправке выводит уведомление и автоматически очищает поля.

## 🛠 Технологии
- **HTML5** (семантическая разметка, атрибуты `required`, `type="tel"`, `label for`)
- **CSS3** (градиенты, состояния `:focus` и `:hover`, плавные `transition`, адаптивная вёрстка)
- **JavaScript** (обработка события `submit`, `event.preventDefault()`, базовая валидация, `alert`)

## 📝 Заметки
- Валидация полей реализована через нативные HTML5-атрибуты (`required`)
- JavaScript перехватывает стандартную отправку формы, предотвращает перезагрузку страницы и показывает `alert`
- Состояния `:focus` на инпутах и `:hover` на кнопке улучшают UX и соответствуют принципам доступности (a11y)
- Архитектура готова к подключению реального бэкенда или сервисов вроде EmailJS / Formspree без переписывания верстки
